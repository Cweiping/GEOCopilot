# Support / 技术支持

## English
- For usage questions, open a GitHub Issue with steps, expected behavior, and screenshots/logs.
- For bug reports, include Chrome version, extension version, and reproduction steps.
- For security issues, please follow `SECURITY.md` and do not disclose publicly first.

## 中文
- 使用问题请提交 GitHub Issue，并附上操作步骤、预期结果、截图/日志。
- Bug 反馈请包含 Chrome 版本、扩展版本和可复现步骤。
- 安全问题请遵循 `SECURITY.md`，避免先公开披露。
