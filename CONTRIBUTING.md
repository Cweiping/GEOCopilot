# Contributing / 贡献指南

## English
1. Fork and create a feature branch.
2. Keep changes focused and documented.
3. Run packaging check:
   ```bash
   bash scripts/package-chrome-extension.sh
   ```
4. Submit PR with summary, motivation, and testing evidence.

## 中文
1. Fork 仓库并创建功能分支。
2. 保持变更聚焦并补充文档。
3. 提交前执行打包检查：
   ```bash
   bash scripts/package-chrome-extension.sh
   ```
4. 提交 PR 时说明变更动机、内容和测试结果。
